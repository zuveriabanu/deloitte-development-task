# Deloitte UX Designer & Developer Task

## Overview
This repository contains the submission for Deloitte's UX Designer and Developer position task.

The project consists of:
- A responsive quiz web application
- Random question selection logic
- Result page with pie chart visualization
- Clean UI implementation based on provided design reference

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
- Chart.js

## How to Run
1. Download or clone the repository.
2. Open `index.html` in a browser.
3. Complete the quiz.
4. View results on the result page.
